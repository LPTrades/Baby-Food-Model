# Version Update - 2026-05-24
## Changes Added

- Rebuilt the Python backtester for the new entry model (FVG + IFVG confirmation).
- Entry now fires at the open of the candle after both FVG and IFVG confirm. No price-return-to-level touch required.
- Both detectors run in parallel, mirroring the Pine state machine: FVG locks on first continuation gap, IFVG tracks the most recent opposite gap and flips on close-through, flip-check runs before the track update.
- Stop = lowest open/close (long) / highest open/close (short) across all 6 gap candles (FVG C1/C2/C3 + IFVG C1/C2/C3). Bodies only, wicks ignored.
- TP changed to 1.5R (was 2R).
- Trade log now records both gaps' candle bodies (o/c pairs) for auditing.
- Vectorized 5m swing detection and swing-level building. Both Python row-by-row loops removed, replaced with pandas shift operations. Near-instant now even on 15 years of data.

## Bugs Fixed

- Removed the old touch-trigger entry path entirely. The old model entered when price returned to the FVG level; that mechanic is gone and was replaced by the open-of-next-bar fill.
- 5m swings now carry a confirmation delay so a swing is only usable after its right-neighbour bar closes. Prevents lookahead from seeing a swing before it could have been known.
- Fixed extremely slow "building swing levels" stage. Cause was two pure-Python loops scanning every 5m bar across the full file. Now vectorized — same output, ~100x faster.

## Reason For Changes

- Match the v1.0 Pine logic so the backtest reflects what the chart actually signals.
- Body-based 6-candle stop ties risk to the two gaps that produced the setup rather than the full displacement leg.
- 1.5R target chosen over 2R.

## Observations

- When both SL and TP fall inside the same 1m bar, the trade resolves as a loss (stop assumed first). Conservative — 1m data can't show intrabar order. One-line change if you want TP-first instead.
- Stop distance varies with gap size now, so risk-per-trade in points is no longer uniform. R is normalised by the 1% risk model so PnL stays comparable, but tight body-stops may get wicked at a higher rate.
- Entry still carries the one-bar-delay slippage from the open fill — worth comparing backtest fills against the confirmation close.
- Smoke-tested on synthetic data only; runs clean end to end. Real stats require your NQ 1m/5m CSVs.
- Swings are detected across all hours on purpose, not just session windows. A swing formed before a session can still be the level price breaks during it, so session filtering happens at break-detection time, not at swing build. This is why the swing count looks high relative to trades.

"""
=============================================================================
NQ FUTURES BACKTESTER  (v1.0 model)
=============================================================================
Matches the v1.0 Pine Script logic exactly:

1. Detect 5m swing highs and lows (immediate neighbours, high/low based)
2. Detect break of most recent swing -> sets mode (long/short), resets setup
3. Run TWO independent 1m gap detectors in parallel off the same break:
     - FVG  : first continuation gap after the break (locks on first one)
     - IFVG : most recent opposite gap that flips on close-through
4. Arm entry when BOTH fvg_found and ifvg_found are true
5. Enter at the OPEN of the NEXT 1m candle (price need NOT touch either gap)
6. Stop  = lowest open/close (long) / highest open/close (short) across all
           6 candles: FVG C1/C2/C3 + IFVG C1/C2/C3.  Bodies only, wicks ignored.
7. TP    = 1.5R from entry, where R = |entry - stop|
8. Close on TP or SL hit — exits are NOT session restricted

Entry sessions (NY time):  03:00-04:00, 09:30-11:00, 14:00-15:00
Break detection is session-gated; exits hold until TP or SL regardless of time.

Outputs:
  - Performance stats (win rate, W/L, avg R, 1%-risk PnL, max DD, streaks)
  - 20 random examples from 2026
=============================================================================
"""

import pandas as pd
import numpy as np
import random
from pathlib import Path

DATA_PATH = r"C:\tradingProject\Data"

# =============================================================================
# CONFIG
# =============================================================================
CONFIG = {
    "account_size": 100000.0,
    "risk_pct":     0.01,       # 1% risk per trade
    "tp_r":         1.5,        # 1.5R take profit
}

# Entry session windows (NY time, 24hr format)
SESSIONS = [
    ("03:00", "04:00"),
    ("09:30", "11:00"),
    ("14:00", "15:00"),
]

# =============================================================================
# SESSION CHECK
# =============================================================================
def in_session(ts: pd.Timestamp) -> bool:
    t = ts.time()
    for start_str, end_str in SESSIONS:
        start = pd.Timestamp(start_str).time()
        end   = pd.Timestamp(end_str).time()
        if start <= t < end:
            return True
    return False

# =============================================================================
# LOAD DATA
# =============================================================================
def load_csv(filename: str) -> pd.DataFrame | None:
    path = Path(DATA_PATH) / filename
    if not path.exists():
        print(f"[ERROR] File not found: {path}")
        return None

    df = pd.read_csv(path)
    df.columns = df.columns.str.strip().str.lower()

    for alias in ["date", "timestamp", "time"]:
        if alias in df.columns and "datetime" not in df.columns:
            df.rename(columns={alias: "datetime"}, inplace=True)

    df["datetime"] = pd.to_datetime(df["datetime"])
    df.set_index("datetime", inplace=True)
    df.sort_index(inplace=True)
    df.dropna(subset=["open", "high", "low", "close"], inplace=True)

    print(f"[LOADED] {filename} | Rows: {len(df)} | {df.index[0]} to {df.index[-1]}")
    return df

# =============================================================================
# SWING DETECTION (5m) — vectorized
# Swing high: high[i] > high[i-1] AND high[i] > high[i+1]
# Swing low:  low[i]  < low[i-1]  AND low[i]  < low[i+1]
# =============================================================================
def detect_swings_5m(df: pd.DataFrame) -> pd.DataFrame:
    df = df.copy()
    h, hp, hn = df["high"], df["high"].shift(1), df["high"].shift(-1)
    l, lp, ln = df["low"],  df["low"].shift(1),  df["low"].shift(-1)

    df["is_swing_high"]    = (h > hp) & (h > hn)
    df["is_swing_low"]     = (l < lp) & (l < ln)
    df["swing_high_price"] = np.where(df["is_swing_high"], h, np.nan)
    df["swing_low_price"]  = np.where(df["is_swing_low"],  l, np.nan)

    df["is_swing_high"] = df["is_swing_high"].fillna(False)
    df["is_swing_low"]  = df["is_swing_low"].fillna(False)
    return df

# =============================================================================
# BUILD A 5m-SWING LOOKUP ALIGNED TO 1m BARS — vectorized
# A 5m swing at bar i is only confirmed after bar i+1 closes (needs the right
# neighbour), so it becomes usable 2 x 5m = 10 min after the swing bar opens.
# We map each confirmed swing to its confirmation time, then as-of merge onto 1m.
# NOTE: swings are detected across ALL hours (not session-filtered) on purpose —
# a swing formed before a session can still be the level that price breaks during
# the session. Session filtering happens at break-detection time, not here.
# =============================================================================
def build_swing_levels(df_5m: pd.DataFrame) -> pd.DataFrame:
    sw = detect_swings_5m(df_5m)
    delay = pd.Timedelta(minutes=10)  # right neighbour (2 x 5m) confirms the swing

    highs = sw[sw["is_swing_high"]].copy()
    highs["conf_time"] = highs.index + delay
    highs["kind"]  = "high"
    highs["price"] = highs["swing_high_price"]

    lows = sw[sw["is_swing_low"]].copy()
    lows["conf_time"] = lows.index + delay
    lows["kind"]  = "low"
    lows["price"] = lows["swing_low_price"]

    out = pd.concat([
        highs[["conf_time", "kind", "price"]],
        lows[["conf_time", "kind", "price"]],
    ]).sort_values("conf_time").reset_index(drop=True)
    return out

# =============================================================================
# MAIN BACKTEST
# =============================================================================
def run_backtest(df_1m: pd.DataFrame, swing_levels: pd.DataFrame, cfg: dict) -> dict:

    o = df_1m["open"].values
    h = df_1m["high"].values
    l = df_1m["low"].values
    c = df_1m["close"].values
    times = df_1m.index
    n = len(df_1m)

    # --- swing levels as a fast as-of pointer ---
    sw_times  = swing_levels["conf_time"].values
    sw_kind   = swing_levels["kind"].values
    sw_price  = swing_levels["price"].values
    sw_ptr    = 0
    n_sw      = len(swing_levels)

    recent_high = None
    recent_low  = None

    mode        = 0       # 1 = long (after low swept), -1 = short (after high swept)

    # FVG state
    fvg_found   = False
    fvg_bodies  = None    # list of (open, close) for the 3 fvg candles

    # IFVG state
    ifvg_found  = False
    pending_gap = False
    gap_edge    = None
    ifvg_bodies = None    # list of (open, close) for the 3 ifvg candles (pre-flip)

    # entry state
    entry_armed = False
    entry_fired = False

    # trade state
    trade_open       = False
    trade_mode       = 0
    trade_entry      = None
    trade_sl         = None
    trade_tp         = None
    trade_entry_time = None
    trade_fvg_bodies = None
    trade_ifvg_bodies= None

    trades       = []
    equity       = cfg["account_size"]
    equity_curve = [equity]

    def reset_setup():
        nonlocal fvg_found, fvg_bodies, ifvg_found, pending_gap, gap_edge
        nonlocal ifvg_bodies, entry_armed, entry_fired
        fvg_found   = False
        fvg_bodies  = None
        ifvg_found  = False
        pending_gap = False
        gap_edge    = None
        ifvg_bodies = None
        entry_armed = False
        entry_fired = False

    for i in range(n):
        ts = times[i]
        active = in_session(ts)

        # ================================================================
        # EXITS — no session restriction, hold until TP or SL
        # If both TP and SL fall inside the same bar's range, assume SL first
        # (conservative).
        # ================================================================
        if trade_open:
            hit_sl = (trade_mode == 1  and l[i] <= trade_sl) or \
                     (trade_mode == -1 and h[i] >= trade_sl)
            hit_tp = (trade_mode == 1  and h[i] >= trade_tp) or \
                     (trade_mode == -1 and l[i] <= trade_tp)

            outcome = None
            exit_px = None
            if hit_sl and hit_tp:
                outcome = "loss"          # conservative: stop first
                exit_px = trade_sl
            elif hit_sl:
                outcome = "loss"
                exit_px = trade_sl
            elif hit_tp:
                outcome = "win"
                exit_px = trade_tp

            if outcome is not None:
                risk_pts = abs(trade_entry - trade_sl)
                r_mult   = cfg["tp_r"] if outcome == "win" else -1.0
                risk_usd = equity * cfg["risk_pct"]
                pnl_usd  = risk_usd * r_mult
                equity  += pnl_usd
                equity_curve.append(equity)

                def _fmt(pair):
                    return f"(o {pair[0]:.2f} / c {pair[1]:.2f})"
                b = trade_fvg_bodies or []
                ib = trade_ifvg_bodies or []
                trades.append({
                    "direction":   "long" if trade_mode == 1 else "short",
                    "outcome":     outcome,
                    "r":           round(r_mult, 2),
                    "entry_time":  str(trade_entry_time),
                    "exit_time":   str(ts),
                    "entry":       round(trade_entry, 2),
                    "sl":          round(trade_sl, 2),
                    "tp":          round(trade_tp, 2),
                    "pnl_usd":     round(pnl_usd, 2),
                    "fvg_c1":      _fmt(b[0]) if len(b) > 0 else "",
                    "fvg_c2":      _fmt(b[1]) if len(b) > 1 else "",
                    "fvg_c3":      _fmt(b[2]) if len(b) > 2 else "",
                    "ifvg_c1":     _fmt(ib[0]) if len(ib) > 0 else "",
                    "ifvg_c2":     _fmt(ib[1]) if len(ib) > 1 else "",
                    "ifvg_c3":     _fmt(ib[2]) if len(ib) > 2 else "",
                })

                trade_open = False
                trade_mode = 0

        # ================================================================
        # UPDATE most-recent confirmed swing levels (as-of this bar's time)
        # ================================================================
        while sw_ptr < n_sw and sw_times[sw_ptr] <= np.datetime64(ts):
            if sw_kind[sw_ptr] == "high":
                recent_high = sw_price[sw_ptr]
            else:
                recent_low = sw_price[sw_ptr]
            sw_ptr += 1

        # ================================================================
        # BREAK DETECTION — session gated (matches Pine: break only in session)
        # ================================================================
        if active:
            if recent_high is not None and h[i] > recent_high and mode != -1:
                mode = -1
                reset_setup()
                recent_high = None
            if recent_low is not None and l[i] < recent_low and mode != 1:
                mode = 1
                reset_setup()
                recent_low = None

        # Need 2 prior bars for any 3-candle gap
        if i < 2:
            continue

        # 3-candle gap test at THIS bar (candles i-2, i-1, i)
        bull_fvg = l[i] > h[i - 2]
        bear_fvg = h[i] < l[i - 2]

        # ================================================================
        # IFVG DETECTOR — session only, mirrors Pine ordering:
        #   STEP 1 flip-check on tracked gap, THEN STEP 2 track newest opposite gap
        # ================================================================
        if active and mode != 0 and not ifvg_found:
            # STEP 1: flip check
            if pending_gap and gap_edge is not None:
                if mode == 1 and c[i] > gap_edge:
                    ifvg_found  = True
                    pending_gap = False
                elif mode == -1 and c[i] < gap_edge:
                    ifvg_found  = True
                    pending_gap = False

            # STEP 2: track newest opposite-side gap (only if not just flipped)
            if not ifvg_found:
                if mode == 1 and bear_fvg:
                    pending_gap = True
                    gap_edge    = l[i - 2]    # top edge: close ABOVE to flip
                    ifvg_bodies = [(o[i - 2], c[i - 2]), (o[i - 1], c[i - 1]), (o[i], c[i])]
                elif mode == -1 and bull_fvg:
                    pending_gap = True
                    gap_edge    = h[i - 2]    # bottom edge: close BELOW to flip
                    ifvg_bodies = [(o[i - 2], c[i - 2]), (o[i - 1], c[i - 1]), (o[i], c[i])]

        # ================================================================
        # FVG DETECTOR — session only, first continuation gap, locks on first
        # ================================================================
        if active and mode != 0 and not fvg_found:
            if mode == 1 and bull_fvg:
                fvg_found  = True
                fvg_bodies = [(o[i - 2], c[i - 2]), (o[i - 1], c[i - 1]), (o[i], c[i])]
            elif mode == -1 and bear_fvg:
                fvg_found  = True
                fvg_bodies = [(o[i - 2], c[i - 2]), (o[i - 1], c[i - 1]), (o[i], c[i])]

        # ================================================================
        # ENTRY TRIGGER — arm when both confirmed, fire on NEXT bar open.
        # Implemented exactly like Pine: if armed already -> fire this bar's open.
        # Entry is session gated (a setup only fires while in session).
        # ================================================================
        if active and not trade_open and not entry_fired:
            if entry_armed:
                # fire on this bar's OPEN
                entry_px = o[i]
                all_bodies = (fvg_bodies or []) + (ifvg_bodies or [])
                body_levels = [v for pair in all_bodies for v in pair]

                if body_levels:
                    if mode == 1:
                        sl_px = min(body_levels)
                        risk  = abs(entry_px - sl_px)
                        if risk > 0:
                            trade_open        = True
                            trade_mode        = 1
                            trade_entry       = entry_px
                            trade_sl          = sl_px
                            trade_tp          = entry_px + cfg["tp_r"] * risk
                            trade_entry_time  = ts
                            trade_fvg_bodies  = fvg_bodies
                            trade_ifvg_bodies = ifvg_bodies
                    elif mode == -1:
                        sl_px = max(body_levels)
                        risk  = abs(entry_px - sl_px)
                        if risk > 0:
                            trade_open        = True
                            trade_mode        = -1
                            trade_entry       = entry_px
                            trade_sl          = sl_px
                            trade_tp          = entry_px - cfg["tp_r"] * risk
                            trade_entry_time  = ts
                            trade_fvg_bodies  = fvg_bodies
                            trade_ifvg_bodies = ifvg_bodies

                entry_armed = False
                entry_fired = True
            elif fvg_found and ifvg_found:
                entry_armed = True

    return {
        "trades":       trades,
        "equity_curve": equity_curve,
        "final_equity": equity,
    }

# =============================================================================
# STATS
# =============================================================================
def compute_stats(results: dict, cfg: dict) -> dict:
    trades = results["trades"]
    if not trades:
        return {"total_trades": 0}

    df = pd.DataFrame(trades)
    wins   = df[df["outcome"] == "win"]
    losses = df[df["outcome"] == "loss"]

    total      = len(df)
    win_count  = len(wins)
    loss_count = len(losses)
    win_rate   = win_count / total if total > 0 else 0
    wl_ratio   = win_count / loss_count if loss_count > 0 else float("inf")
    avg_r      = df["r"].mean()
    total_pnl  = df["pnl_usd"].sum()

    eq   = np.array(results["equity_curve"])
    peak = np.maximum.accumulate(eq)
    dd   = (eq - peak) / peak * 100
    max_dd = dd.min()

    outcomes        = df["outcome"].tolist()
    max_win_streak  = 0
    max_loss_streak = 0
    cur_win         = 0
    cur_loss        = 0
    for o in outcomes:
        if o == "win":
            cur_win  += 1
            cur_loss  = 0
            max_win_streak = max(max_win_streak, cur_win)
        else:
            cur_loss += 1
            cur_win   = 0
            max_loss_streak = max(max_loss_streak, cur_loss)

    return {
        "total_trades":        total,
        "wins":                win_count,
        "losses":              loss_count,
        "win_rate_%":          round(win_rate * 100, 2),
        "w/l_ratio":           round(wl_ratio, 2),
        "avg_r":               round(avg_r, 3),
        "total_pnl_usd":       round(total_pnl, 2),
        "final_equity":        round(results["final_equity"], 2),
        "max_drawdown_%":      round(max_dd, 2),
        "longest_win_streak":  max_win_streak,
        "longest_loss_streak": max_loss_streak,
    }

# =============================================================================
# PRINT STATS
# =============================================================================
def print_stats(stats: dict):
    print("\n" + "=" * 55)
    print("  BACKTEST RESULTS")
    print("=" * 55)
    for k, v in stats.items():
        print(f"  {k.replace('_', ' ').title():<28} {v}")
    print("=" * 55)

# =============================================================================
# PRINT 20 RANDOM 2026 EXAMPLES
# =============================================================================
def print_examples(trades: list, n: int = 20):
    trades_2026 = [t for t in trades if str(t["entry_time"]).startswith("2026")]

    if not trades_2026:
        print("\n[INFO] No trades found in 2026.")
        return

    sample = random.sample(trades_2026, min(n, len(trades_2026)))
    sample.sort(key=lambda x: x["entry_time"])

    print(f"\n{'=' * 90}")
    print(f"  {len(sample)} RANDOM EXAMPLES FROM 2026")
    print(f"{'=' * 90}")

    for i, t in enumerate(sample, 1):
        direction = t["direction"].upper()
        outcome   = t["outcome"].upper()
        print(f"\n  [{i}] {direction} | {outcome} | R: {t['r']}")
        print(f"      Entry Time : {t['entry_time']}")
        print(f"      Exit Time  : {t['exit_time']}")
        print(f"      Entry Price: {t['entry']:.2f}")
        print(f"      Stop Loss  : {t['sl']:.2f}")
        print(f"      Take Profit: {t['tp']:.2f}")
        print(f"      Risk       : {abs(t['entry'] - t['sl']):.2f} pts")
        print(f"      FVG  bodies: {t['fvg_c1']} {t['fvg_c2']} {t['fvg_c3']}")
        print(f"      IFVG bodies: {t['ifvg_c1']} {t['ifvg_c2']} {t['ifvg_c3']}")
        print(f"      PnL        : ${t['pnl_usd']:,.2f}")

    print(f"\n{'=' * 90}")

# =============================================================================
# MAIN
# =============================================================================
if __name__ == "__main__":
    print("\n" + "=" * 55)
    print("  NQ FUTURES BACKTESTER  (v1.0 model)")
    print("=" * 55)
    print(f"\n  Entry: open of candle after FVG + IFVG both confirm")
    print(f"  Stop : lowest/highest body of all 6 gap candles")
    print(f"  TP   : {CONFIG['tp_r']}R")
    print(f"\n  Entry sessions (NY time):")
    for s, e in SESSIONS:
        print(f"    {s} - {e}")
    print(f"  Exits: hold until TP or SL (no time restriction)\n")

    file_1m = input("Enter 1m CSV filename (e.g. NQ_1m.csv): ").strip()
    file_5m = input("Enter 5m CSV filename (e.g. NQ_5m.csv): ").strip()

    df_1m = load_csv(file_1m)
    df_5m = load_csv(file_5m)

    if df_1m is None or df_5m is None:
        print("[FATAL] Could not load data. Check filenames and path.")
        exit()

    print("\n[RUNNING] Building swing levels...")
    swing_levels = build_swing_levels(df_5m)
    print(f"[INFO] {len(swing_levels)} confirmed 5m swings.")

    print("[RUNNING] Backtesting — this may take a moment...")
    results = run_backtest(df_1m, swing_levels, CONFIG)

    if not results["trades"]:
        print("[DONE] No trades generated. Check data or logic.")
        exit()

    stats = compute_stats(results, CONFIG)
    print_stats(stats)
    print_examples(results["trades"], n=20)

    out = Path(DATA_PATH) / "backtest_results_v1.csv"
    pd.DataFrame(results["trades"]).to_csv(out, index=False)
    print(f"\n[EXPORT] Full trade log saved to: {out}")
